# Flash Cards JavaScript

Aplicação de Flash Cards desenvolvida com React para ajudar no estudo de conceitos de JavaScript através de perguntas e respostas interativas.

Este projeto foi desenvolvido como parte do desafio de React da plataforma roadmap.sh.

## Funcionalidades

- Flash cards pré-definidos
- Sistema de perguntas e respostas
- Mostrar e esconder respostas
- Navegação entre os cards
- Barra de progresso dinâmica
- Interface responsiva e minimalista

## Tecnologias

- React
- JavaScript
- CSS3
- Vite

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
